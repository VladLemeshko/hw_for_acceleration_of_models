# Анализ и оптимизация нейросетевой модели BERT

## Описание проекта

Данный репозиторий содержит аналитический отчёт по исследованию возможностей ускорения модели **BERT (Bidirectional Encoder Representations from Transformers)** для задачи классификации текстовых запросов в онлайн-сервисе технической поддержки.

    └── analytical_report.md       # Краткий аналитический отчёт

## Выбранная модель

**BERT-base-uncased** — трансформерная модель для обработки естественного языка, применяемая в задаче классификации текстовых обращений пользователей в системе технической поддержки.

## Ключевые характеристики сценария

| Параметр | Значение |
|----------|----------|
| Задача | Классификация текста (multi-class) |
| Режим работы | Онлайн-инференс |
| Целевая платформа | Серверный GPU (NVIDIA T4) |
| Требуемая латентность | ≤ 50 мс (p99) |
| Пропускная способность | ≥ 1000 RPS |

## Основные выводы

1. **Главные bottlenecks**: механизм self-attention с квадратичной сложностью по длине последовательности, матричные умножения в FFN-слоях
2. **Рекомендуемые методы ускорения**: INT8 квантизация, дистилляция в DistilBERT, оптимизация через TensorRT
3. **Ожидаемое ускорение**: 2-4x при сохранении >95% исходного качества
